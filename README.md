# IS2SaVe

This is a git lfs project. Make sure you have git lfs installed.

## Prerequisites

- [PostgreSQL](https://www.postgresql.org/) running on port `5432`
- [Grafana](https://grafana.com/) running on port `3000`
- [AnyLogic](https://www.anylogic.com/) (PLE is sufficient)
- Python, NodeJS, npm

## Getting started

Start AnyLogic & load `DSS2SaVe.original.alp`

In AnyLogic: open the .. and change the database credentials according to your configuration

Run the experiment _(will start the simulation server)_

Go to the main directoriy and run **`sh run.sh`** from your terminal _(will do everything for you automatically)_

Open [http://localhost:2020](http://localhost:2020) to view it in your browser.
